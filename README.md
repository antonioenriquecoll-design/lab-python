# Laboratorio Python HPC-Bio-Mendel

Este repositorio contiene los materiales de la práctica 4:

- **reduc-operation-alumno05.ipynb**: notebook principal con operaciones de reducción y resultados documentados en SLURM.  
- **reduc-operation-alumno05.py**: versión exportada del notebook para ejecución en la cola mediante el script SLURM.  
- **pi-solution-alumno05.ipynb**: actividad extra de cálculo de π mediante Monte Carlo.  
- **submit_Python_hpc-bio-mendel-alumno05.sh**: script SLURM para ejecutar en la cola *hpc-bio-mendel*.  
- **Informe-Lab4.pdf**: informe con aspectos positivos y negativos de la práctica.  
- **README.md**: descripción del repositorio y sus contenidos.
